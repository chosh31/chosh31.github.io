                             System.out.println("name = "    + subElement1.item(0).getNodeValue() + ", "

                                                     //System.out.println("nameElement=" + headLineElement.getElementsByTagName("nameElement"));

아울러 XML의 파싱 방법에는 SAX와 DOM 파싱 방법이 있다는 것을 지식in을 통해서 알게 되었당 ㅋㅋ

http://kin.naver.com/qna/detail.nhn?d1id=1&dirId=1040201&docId=122082370&qb=amF2YSB4bWwg7YyM7Iux&enc=utf8&section=kin&rank=1&search_sort=0&spq=0&pid=gIqQyc5Y7tosssL26zsssc--436865&sid=T6qfyHOGqk8AAAN2ESU


XML은 크게 두 가지 방법으로 처리를 합니다.
SAX 계열 파서와 DOM 계열 파서를 이용합니다.

DOM 계열 파서는 XML 문서를 고스란히 메모리에 올려놓고 처리하는 방식이고,
SAX 계열은 XML 문서를 스트림 방식으로 읽으면서 요소를 만날 때마다 이벤트 방식으로 처리를 합니다.

DOM 파서는 메모리 상에 문서가 존재하는 동안에는 얼마든지 전체를 탐색하며 처리를 할 수 있지만,
SAX 파서는 스트림의 특성상 문서를 한 번 읽로 지나가면 뒤로 돌릴 수가 없습니다.

DOM 파서는 XML 문서의 크기만큼 메모리 공간이 요구되지만,
SAX 파서는 문서의 크기에 상관 없이 비교적 적은 메모리 만으로도 처리가 가능합니다.
그래서 대용량 XML 문서의 경우 DOM 보다는 SAX를 이용해서 처리합니다.
