Tomcat 라이브러리 이클립스 추가 방법 TIp / Programming

2012/09/07 09:39

http://blog.naver.com/phanta03/140167532904

전용뷰어로 보기 포스트를 근사하게 보는 방법





보통은 tomcat을 깔면 이클립스에서 자동으로 톰캣 라이브러리를 땡겨오지만 이것이 안되는 경우도 종종 있었다.


그래서 tomcat 라이브러리 중 servlet 클래스를 사용해야되는데 톰캣 라이브러리가 인식되지 않아 통째로 복사해서 jre 폴더에 


뒤집어 씌우고 그랬었는데 이러니 라이브러리끼리 충돌이 일어난다던가 톰캣을 사용하지 않는 환경에서 돌렸을 때 뭐 듣도보도 못한 에러


메세지가 나를 반겼다. 


그러므로 다 알지만 누군가는 모를 이클립스에 톰캣 라이브러리 추가방법에 대해서 간략 설명 한다. ( 사실 내가 안 까먹기 위해서다. 다른 사람한테 설명하기도 귀찮아서  )


이미지 파일을 캡쳐하려 해봤는데 지금 포스팅하는 곳에선 캡쳐가 불가하단다 ㅠ 


간단한 것이니 그냥 글로  진행하자


===========================================================================================================================


1. 프로젝트에서 오른 클릭해서 build path(빌드 경로) /  build path config(빌드 경로 구성) 로 들어간다.


2. library add(라이브러리 추가) (현재 쓰고 있는 이클립스가 한글버전이라 영문버전에선 어찌되어 있을지 모르겠다.)


3. server runtime(서버 런타임) 클릭 / 확인 


4. 자신이 추가하고자 하는 톰캣 지정 (pc에 톰캣이 여러개 깔려있을 경우는 여러개가 뜰 수도 있다. 보통은 자신이 쓰는 것 하나만 뜬다.)


5. next(다음) 누르고 확인 누르면 완료


6. 프로젝트 탐색기에서 라이브러리를 펼쳐보면 톰캣 라이브러리가 추가되어 있는 것을 확인할 수 있다.




===========================================================================================================================
